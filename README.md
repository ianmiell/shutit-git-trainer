To run as a trainer/coaching application:

```
pip install shutit
git clone https://github.com/ianmiell/shutit-git-trainer
cd shutit-git-trainer/bin
./build.sh --trainer
```

Or a walkthrough:

```
pip install shutit
git clone https://github.com/ianmiell/shutit-git-trainer
cd shutit-git-trainer/bin
./build.sh --walkthrough
```
